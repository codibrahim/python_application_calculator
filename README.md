# 🧮 Python Application Calculator

A simple command-line calculator built with Python. Supports basic arithmetic operations.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⚠️ Division by zero protection
- 🔁 Continuous input loop until user exits

---

## 🚀 How to Run

### Requirements

- Python 3.x installed on your machine

### Steps

**1. Clone the repository:**
```bash
git clone git@github.com:codibrahim/python_application_calculator.git
```

**2. Navigate to the project folder:**
```bash
cd python_application_calculator
```

**3. Run the calculator:**
```bash
python3 calculator.py
```

---

## 💻 Usage Example

```
=== Python Calculator ===
Operations: add, subtract, multiply, divide
Type 'exit' to quit

Enter operation: add
Enter first number: 10
Enter second number: 5
Result: 15.0

Enter operation: divide
Enter first number: 10
Enter second number: 0
Result: Error: Division by zero is not allowed.

Enter operation: exit
Goodbye!
```
